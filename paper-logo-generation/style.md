# 论文 Logo 风格指南

这个 Skill 面向论文宣传图、项目发布图、社交媒体封面和技术品牌化场景，目标是把论文内容压缩成一个更像“项目 Logo 提案”的图片 Prompt。

## 核心视觉方向

- 固定以一只动漫风橘猫作为画面核心。
- 构图要干净、极简、轻量，偏手绘封面感而不是复杂海报感。
- 优先追求白底或浅米色纸张底、明确重心和较少杂讯。
- 最终气质应更接近“学术感手绘项目封面”，而不是完整场景插画。

## Prompt 必备元素

每个最终 Prompt 应包含：

1. 论文最核心的技术主题。
2. 2 到 4 个提炼后的关键词。
3. 固定橘猫主角及其桌面学习场景。
4. 将关键词放入纸上的算法草图、逻辑流程图和符号结构中。
5. 明确的成图风格约束与负向限制。

## 美术规则

- 采用浅水彩与彩铅混合的手绘风格。
- 主角是一只圆润、可爱、橘色的猫，大圆眼睛，白色爪子，蓝色项圈和黄色铃铛。
- 猫坐在简洁书桌前，手持铅笔，表情专注，正在解题或绘制逻辑图。
- 允许少量柔和细节，但整体必须高度简化。
- 要有明显留白、纸张质感和清晰外轮廓。
- 背景尽量简单，只保留少量淡淡的几何线条或安静学习环境暗示。
- 顶部允许出现手绘文字 `SKILLEVO`，除此之外不要出现其他文字。
- 不要水印，不要 UI 面板，不要截图感，不要复杂场景。

## 最终 Prompt 的语气

最终 Prompt 应该像一段可直接投喂给图像模型的专业视觉说明：

- 具体
- 有画面感
- 简洁但鲜明
- 一次复制即可用于 Nano Banana 2
