# Prompt 模板参考

最终输出建议使用下面的结构：

```md
# <论文标题>

## Keywords
- 关键词 1
- 关键词 2
- 关键词 3

## Mascot
- animal: 橘猫
- rationale: 固定参考主角，用论文关键词区分不同论文视觉内容

## Prompt
<最终可直接用于 Nano Banana 2 的 Prompt>
```

写 Prompt 时建议遵循：

- 先明确固定的橘猫主角与桌面学习场景。
- 把论文贡献转成纸上的算法草图、逻辑流程图和符号线索。
- 背景保持白色或浅米色纸张质感。
- 强调“极简手绘、水彩、彩铅、学术感、专注感”。
- 如果 Skill 能可靠提取到论文标题，就把标题写在顶部；如果没有标题，就省略顶部文字。
- 结尾补上约束，例如：`不要额外文本、不要水印、不要 UI、不要复杂背景`。
