# Prompt 模板参考

最终输出建议使用下面的结构：

```md
# <论文标题>

## Keywords
- 关键词 1
- 关键词 2
- 关键词 3

## Mascot
- animal: <动物名称>
- rationale: <一句话说明原因>

## Prompt
<最终可直接用于 Nano Banana 2 的 Prompt>
```

写 Prompt 时建议遵循：

- 先明确动物主角。
- 把论文贡献转成一个清晰的视觉隐喻。
- 背景保持简洁。
- 强调 Logo 感、标志感、精致插画质感。
- 结尾补上约束，例如：`不要文字、不要字母、不要水印、不要 UI、透明底或干净背景`。
