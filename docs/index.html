GitHub Pages 展示页

让每篇论文，都拥有一枚更像项目品牌的 Logo。

这是一个专门为研究论文设计的 Prompt Skill 页面。 它从摘要与贡献出发，抽取关键词、匹配动物主角， 最终产出一段可以直接用于 Nano Banana 2 的 Logo Prompt。

摘要 贡献 关键词 Prompt

读取论文

支持 PDF、论文链接，以及包含 PDF 的本地项目目录。

提炼主题

聚焦摘要和核心贡献，而不是生成泛泛的技术总结。

生成 Logo Prompt

输出包含动物主角、视觉隐喻和负向约束的最终描述。

项目特性

它做的不是“帮你画图”，而是先帮论文建立视觉身份。

偏品牌，不偏模板

它会把方法气质、研究方向和传播用途一起考虑， 让最终 Prompt 更像项目 Logo 概念，而不是随手生成的插画描述。

偏识别度，不偏堆砌

它强调单主体、强轮廓、简单背景和高辨识度， 让图像结果更适合封面、社媒图和项目发布视觉。

四步工作流

从论文内容到成品 Prompt，路径尽量短，也尽量稳。

01

读取摘要

抓住研究对象、任务目标和方法核心。

02

读取贡献

优先寻找最能代表论文价值的贡献段落。

03

选择主角

根据领域气质映射到合适的动漫动物形象。

04

输出 Prompt

得到可直接用于 Nano Banana 2 的最终描述文本。
